# Mirza Creative — SEO Optimized GitHub Pages Website

Ready-to-upload version for GitHub Pages.

## Upload steps
1. Open your GitHub repository.
2. Delete old files or replace them with this ZIP content.
3. Commit changes.
4. Go to Settings → Pages and keep branch as `main` / root.
5. After live update, submit this sitemap in Google Search Console:
   `https://mirzacreative.github.io/mirzacreative/sitemap.xml`

## Included SEO files
- `robots.txt`
- `sitemap.xml`
- `image-sitemap.xml`
- `site.webmanifest`
- `llms.txt`
- `.nojekyll`
- `404.html`
- JSON-LD structured data on pages
- canonical, OpenGraph and Twitter metadata
- fixed language markup
- Apple touch icon

## Note
GitHub Pages cannot force true server-side 301 redirects for `/index.html` or `www/non-www` on the default github.io domain. When you buy a custom domain, use Cloudflare or your domain DNS/CDN to set 301 redirects and then update canonical URLs to the custom domain.
